#06.Eclipse快捷键

##1. 基本快捷键

 **复制** ：CTRL + C
 
 **粘贴** ：CTRL + V 
 
 **剪切** ：CTRL + X
 
 **查找** ：CTRL + F
 
 **撤销** ：CTRL + Z
 
 **重做** ：CTRL + Y  <code>重做上一步被撤销的操作</code>
 
**注释** ：CTRL + /
 
 **提示** ：ALT + /  <code>显示代码提示</code>
 
 **格式化代码** ：CTRL + SHIFT + F 
 
 **标签页切换** ：CTRL + (PageUp | PageDown)
 
 **查找文件或字符** ：CTRL + H
