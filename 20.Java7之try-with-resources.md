#20.Java7之try-with-resources

##20.1 释放资源：Java7之前的做法

在Java7之前，你需要在`finally`语句块中释放资源而不管`try`语句块正常结束抑或中断抛出异常。例如：

```Java
BufferedReader br = new BufferedReader(new FileReader(path));
try {
  return br.readLine();
} finally {
  if (br != null) br.close();
}
```
